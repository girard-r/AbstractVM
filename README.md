# AbstractVM

AbstractVM est une machine a pile capable de calculer des expressions arithmetiques simples. Les expressions arithmetiques sont donnees a la machine sous forme de programmes assembleur simples.
Exemple :

push int32(42)

push int32(33)

add

push float(44.55)

mul

push double(42.42)

push int32(42)

dump

pop

assert double(42.42)

exit
